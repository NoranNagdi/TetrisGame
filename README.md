# 🎮 Tetris Game (WPF + C# | MVVM)

A classic **Tetris game** built using **WPF** and **C#**, following the **MVVM (Model-View-ViewModel)** architectural pattern.  
This project demonstrates clean separation of concerns, responsive UI design, and well-structured game logic.

---

## 🚀 Features

- 🧱 Classic Tetris gameplay  
- 🎯 Score tracking  
- ⚡ Smooth piece movement and rotation  
- 🧠 MVVM architecture for maintainability  
- 🎨 WPF-based responsive UI  
- ⬆️ Instant piece drop  
- 🔄 Restart game option  

---

## 🛠️ Technologies Used

- **C#**
- **WPF (Windows Presentation Foundation)**
- **MVVM Design Pattern**
- **.NET**

---

## 🎮 Game Controls

| Key            | Action                          |
|----------------|---------------------------------|
| ⬅️ Left Arrow  | Move piece left                 |
| ➡️ Right Arrow | Move piece right                |
| ⬇️ Down Arrow  | Move piece down faster          |
| ⬆️ Up Arrow    | Drop piece instantly            |
| R              | Rotate the current piece        |

---

## 🧩 How to Play

- Control falling blocks and position them to complete horizontal lines  
- Completing a full line clears it and increases your score  
- The game ends when the pieces stack up to the top of the board  

---

## 🏗️ Project Structure

The project follows the **MVVM pattern**:

- **Model** → Game logic, grid system, and piece definitions  
- **View** → UI components (XAML)  
- **ViewModel** → Data binding, commands, and game state management
